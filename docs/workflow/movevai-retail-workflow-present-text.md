# สคริปต์นำเสนอ Workflow ระบบ MoveVai Retail

ภาพนี้คือภาพรวมการทำงานของระบบ MoveVai Retail ตั้งแต่รับออเดอร์เข้ามา ตรวจสอบข้อมูล จัดส่งสินค้า ไปจนถึงแอดมินปิดงาน โดยมีจุดสำคัญคือ ลูกค้าสามารถติดตามสถานะการจัดส่งได้ด้วยตัวเองผ่านลิงก์หรือ QR

ขั้นแรก ออเดอร์สามารถเข้ามาได้หลายช่องทาง เช่น LINE แชตภายใน ไฟล์หรือสคริปต์ และการคีย์มือ หลังจากนั้นข้อมูลจะถูกส่งเข้าสู่ Order Inbox เพื่อให้แอดมินตรวจสอบความถูกต้องของลูกค้า ที่อยู่ และวิธีการจัดส่งก่อนปล่อยงาน

เมื่อข้อมูลพร้อม ระบบจะแยกเส้นทางการจัดส่งออกเป็น 2 แบบ คือการมอบหมายให้คนขับภายใน และการส่งผ่านไปรษณีย์ไทย ถ้าเป็นงานคนขับ แอดมินจะจัดคิว วางรอบ และส่งงานเข้าแอป Messenger ของคนขับ ส่วนงานไปรษณีย์ไทยจะ export รายการ ใส่เลขพัสดุ และส่งมอบให้ไปรษณีย์ดำเนินการต่อ

ในฝั่งคนขับ เมื่อเริ่มจัดส่ง แอป Messenger จะช่วยนำทางและเก็บหลักฐาน เช่น GPS รูปถ่าย ลายเซ็น หรือ OTP จากนั้นแอดมินจะตรวจหลักฐานและเป็นผู้ปิดงาน หากส่งไม่สำเร็จ ระบบยังรองรับการนัดส่งใหม่ การส่งกลับ หรือการปิดเคสตามสถานะจริง

ระหว่างกระบวนการนี้ ลูกค้าไม่ต้องรอสอบถามแอดมินอย่างเดียว เพราะสามารถเปิดลิงก์หรือ QR เพื่อติดตามสถานะได้เอง เห็นลำดับตั้งแต่รับออเดอร์ นัดวันจัดส่ง กำลังจัดส่ง ไปจนถึงส่งสำเร็จ พร้อมข้อมูลที่จำเป็น เช่น ตำแหน่งคนขับสดและหลักฐานการส่ง โดยระบบจะแสดงเฉพาะข้อมูลที่เหมาะสมต่อความเป็นส่วนตัว

สรุปคือ workflow นี้ช่วยให้แอดมินควบคุมงานได้ชัด คนขับทำงานจากแอปเดียว และลูกค้ามองเห็นความคืบหน้าการจัดส่งได้ตลอดทาง
