# Production Notes — สิ่งที่ต้องจัดหา/อัปเกรด

รายการ infra ที่ตอนนี้ยังใช้ของ **free / open-source** ไปก่อน (dev/demo) — ยังไม่สะท้อนต้นทุน production จริง ต้องจัดหา/อัปเกรดก่อนขึ้นจริง

| #   | รายการ                    | สถานะตอนนี้                               | ต้องทำ                                                                                                             |
| --- | ------------------------- | ----------------------------------------- | ------------------------------------------------------------------------------------------------------------------ |
| 1   | **Google Maps API**       | ใช้ open-source (OSM / OSRM) แทน          | ซื้อ/เปิดใช้ Google Maps API จริง (public OSRM ทำระยะทางเพี้ยนบน native)                                           |
| 2   | **Cloud เก็บรูปภาพ**      | ยังไม่มี cloud storage                    | จัดหา cloud storage/bucket สำหรับเก็บรูป (POD photo, รูปต้นฉบับ OCR import)                                        |
| 3   | **Typhoon API (OCR)**     | ใช้ tier **free** และ **เกิน limit** แล้ว | อัปเกรดเป็น paid plan (engine ที่ backend ใช้ parse รูป LINE import → ฟิลด์) + ข้อจำกัด เกิน limit file ใหญ่เกินไป |
| 4   | **Hardware track สินค้า** | ยังไม่มี                                  | ถ้าต้องการ track ตัวสินค้าจริง (นอกจาก track คนขับ) ต้องซื้ออุปกรณ์ track มาเพิ่มทีหลัง                            |
